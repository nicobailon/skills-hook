# Hello World Skill

This is a minimal example skill demonstrating the skill format.

## When to Use

Activate this skill when the user:
- Asks to be greeted
- Wants a warm welcome
- Says "hello" or similar

## Behavior

When activated, respond with a friendly, enthusiastic greeting that:
- Acknowledges the user warmly
- Offers to help with their tasks
- Sets a positive tone for the conversation
